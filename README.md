# FinanceFlow

A Personal Finance Tracker built using Flask and SQLite.

## Features
- User Signup/Login
- JWT Authentication
- Income Management
- Expense Tracking
- Dashboard Analytics
- PDF Report Generation

## Tech Stack
- Python
- Flask
- SQLite
- HTML/CSS
- JWT
- Render Deployment
